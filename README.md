Este repositorio eh basicamente meu portfolio mostrando o que eu trabalho e aprendo na area de cybersecurity. Tenho experiencia com analise de logs, hardening de sistemas e um interesse crescente em cloud security, especialmente na AWS.

Meu foco ta em:
- Analisar logs e eventos de segurança

- Resposta a incidentes basicos
